# XYチャート（xychart-beta）

> ⚠️ **beta構文**: 表示環境によってレンダリングされない場合があります。使用前に確認してください。

## 概要

X軸・Y軸を持つ棒グラフ・折れ線グラフを表現する図。時系列データや比較データの可視化に使う。

## 使いどころ

- 時系列データの推移（月別売上、件数など）
- カテゴリ間の数値比較
- 複数系列の重ね合わせ（棒 + 折れ線）

## 使わないケース

- 構成比 → `pie`
- 2軸の分類 → `quadrantChart`
- 量の流れ → `sankey-beta`

---

## 基本テンプレート

```
xychart-beta
    title "チャートのタイトル"
    x-axis [ラベル1, ラベル2, ラベル3]
    y-axis "Y軸ラベル" 最小値 --> 最大値
    bar [値1, 値2, 値3]
    line [値1, 値2, 値3]
```

`bar` と `line` は組み合わせて使える。

---

## 実例

### 例1: 月別バグ件数の推移

```mermaid
xychart-beta
    title "月別バグ件数"
    x-axis [1月, 2月, 3月, 4月, 5月, 6月]
    y-axis "件数" 0 --> 50
    bar [20, 35, 28, 15, 10, 8]
    line [20, 35, 28, 15, 10, 8]
```

### 例2: ドメイン別テストカバレッジ

```mermaid
xychart-beta
    title "ドメイン別テストカバレッジ（%）"
    x-axis [注文, 在庫, 決済, 通知, 認証]
    y-axis "カバレッジ" 0 --> 100
    bar [85, 72, 90, 60, 95]
```
